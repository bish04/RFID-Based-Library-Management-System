# Abstract

The RFID-Based Smart Library Management System is an Internet of Things (IoT) based solution designed to automate library operations such as book issue and return. Traditional library systems rely heavily on manual record maintenance, which is time-consuming and prone to human error.

This project utilizes an ESP32 microcontroller, RC522 RFID reader, Firebase Realtime Database, Google Sheets, and an I2C LCD display to provide a smart and efficient library management system. Each student and book is assigned a unique RFID tag. The system verifies student identities, checks book availability, updates inventory records, and logs all transactions in real time.

The integration of Firebase enables cloud-based storage and retrieval of information, while Google Sheets provides an easily accessible transaction log. The proposed system reduces manual effort, improves accuracy, and provides a scalable solution for educational institutions.

Keywords:
RFID, ESP32, Firebase, IoT, Smart Library, Google Sheets, Automation.
